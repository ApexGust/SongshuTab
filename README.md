## 松鼠标签

面向 Chrome/Edge 的 MV3 插件，用于把当前窗口的标签页一键收纳到分组，支持在侧栏或新标签页里轻量查看、重命名、拖拽调整与恢复。

## 说明
这个插件初衷来源于OneTab, OneTab我用了很多年，很好用，自身轻量，界面清爽，市场当然还有很多标记管理插件，试过几个感觉太重了，只是日常管理几个常用标签页，不需要那么复杂。
OneTab就是这样，足够简单。不过有些功能不满足我的需求，所以自己实现一个。主要做了以下扩展
1. 默认快捷键ctrl/command+e, 收纳当前活动标签到「待领养标签」快速组。
2. 默认给了一个常驻的标签组「标签钉子户」，这个组里打开链接之后也不会被消失，一直存在。
3. 支持给链接重命名，特别是网址一样后面参数不同的链接，可以重命名，方便区分。
4. 支持用户自建分组，支持新增、重命名、拖拽排序、清空、删除（非默认组），用户自建分组可配置打开链接后是否在列表中保留。
5. 支持深色/浅色主题切换。
6. 支持浏览器侧栏（默认）或新标签页展示面板，可在设置中切换。
7. 支持快捷键命令，可以在浏览器「快捷键」页面调整。
8. 点击分组图标可以快速展开/收起分组。

### 功能特性
- 收纳模式：一键收纳当前窗口全部标签，或通过快捷键收纳当前活动标签到「待领养标签」快速组。
- 侧栏/标签页视图：支持浏览器侧栏（默认）或新标签页展示面板，可在设置中切换。
- 分组管理：默认「标签钉子户」持久组与「待领养标签」临时组，支持新增、重命名、拖拽排序、清空、删除（非默认组）。
- 标签操作：重命名、删除、拖拽跨组移动、点击恢复单个或整组标签；非持久组在恢复后会自动清空。
- 主题与持久化：深色/浅色主题切换；用户自建分组可配置点击后是否保留。

### 快速开始（开发/调试）
1) 打开 Chrome/Edge 扩展管理页：`chrome://extensions/`。  
2) 打开「开发者模式」。  
3) 点击「加载已解压的扩展程序」，选择本仓库目录 `/Users/prayfff/Downloads/SongshuTab`。  
4) 安装后点击扩展图标，按当前设置在侧栏或新标签页打开面板。

### 使用指南
- 收纳当前窗口全部标签：点击扩展图标，然后在面板中查看生成的分组。
- 快速收纳当前活动标签：默认快捷键 `Ctrl+E`（macOS 为 `Command+E`），会将当前活动标签移入「待领养标签」并关闭该标签。
- 恢复标签：在面板点击标签标题可单个恢复；点击「全部打开」可批量恢复分组（非持久组恢复后会清空）。
- 分组操作：点击「新增标签组」，双击组名或标签名可重命名，拖拽标签在组间移动，点击「清空组」或「删除组」（仅用户创建的组）。

### 设置
- 入口：面板右上角齿轮或扩展详情页的「扩展选项」。
- 配置项：
  - 主题：深色 / 浅色。
  - 展示方式：侧栏（默认，需浏览器支持 sidePanel 权限）或新标签页。
  - 用户分组持久化：为各自建分组设置「保留/不保留」行为。
  - 快捷键：默认 `Ctrl/Command + E`，可在浏览器「快捷键」页面调整。
